About

XSEDE Information Services (IS) provides a standard set of information browsing pages and information access APIs so that developers and software in the extended XSEDE community can access information about XSEDE resources, use cases, software, services, science gateways, and other e-infrastructure elements. XSEDE user documentation, user portal (XUP), and monitoring systems access information services. IS content comes from the XSEDE databases and XSEDE HPC/HTC/storage/... resource service providers.

Information Access

  • Warehouse API documentation (swagger) (REST/HTTP)
  • tginfo command line discovery client (CLI)
